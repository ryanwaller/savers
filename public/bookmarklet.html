Savers Bookmark

Click the bookmark on any page to save it to Savers.

  1. Click Copy save URL above.
  2. Bookmark this page (Ctrl+D / ⌘+D).
  3. Right-click the new bookmark → Edit.
  4. Paste the copied URL and name it Save to Savers.

Can’t see your bookmarks bar? Press Ctrl+Shift+B (Windows) or ⌘+Shift+B (Mac).

How does this work?

The bookmark opens a Savers page that reads which page you came from and saves it. A new tab opens for a moment, then closes. For a more seamless experience with additional options (tags, collection picker, custom title), use the inline bookmarklet from Settings → Bookmarklet.
