Set up your quick-save bookmark

Copy the bookmarklet code, bookmark this page, then paste the code into that bookmark.

Open Savers
1
Click Copy bookmarklet code. This copies the code your bookmark will use.
2
Bookmark this page (Ctrl+D / ⌘+D). That captures the Savers icon in your bookmarks bar.
3
Right-click the new bookmark and choose Edit. Open the bookmark details so you can replace its URL.
4
Paste the copied code and name it Save to Savers. After that, a normal left-click opens Savers and saves the page you were on.
How it works

Opens Savers in a popup window so your login session works. Nothing to install — just a bookmark. Popups must be allowed for Savers (most browsers auto-allow clicks).
