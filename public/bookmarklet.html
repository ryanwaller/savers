Quick save setup

Save any page without leaving your bookmarks bar.

Set up a simple quick-save bookmark once, then use it from any browser to send the current page straight into Savers.

Works in any browser Keeps the Savers icon Best fallback when the extension is unavailable
Set up your quick-save bookmark

Copy the Savers save link, bookmark this page, then paste the link into that bookmark.

Open Savers
1
Click Copy save URL. This copies the link your bookmark will use.
2
Bookmark this page (Ctrl+D / ⌘+D). That captures the Savers icon in your bookmarks bar.
3
Right-click the new bookmark and choose Edit. Open the bookmark details so you can replace its URL.
4
Paste the copied link and name it Save to Savers. After that, a normal left-click saves the page you were on and returns you right back.
How it works

Left-clicking the bookmark briefly opens Savers, captures the page you came from, saves it, and sends you right back. Because the bookmark points to a normal link instead of javascript:, Chrome keeps the Savers icon intact. For a richer inline save flow with suggestions and collection controls, use Settings → Save to Savers.
